# Python course task

В данном репозитории размещено решение задания из Темы 7 курса Основы Python.

## Описание задачи

Программа принимает список, находит самую длинную строку,
а затем определяет в ней самую длинную подстроку без повторяющихся символов.

## Функции

- check_input — проверяет, является ли элемент строкой
- get_substring — находит самую длинную подстроку без повторений
- main — управляет логикой программы и выводит результат
